# INSTALLATION

```bash
sudo apt install gstreamer-1.0
sudo apt install gstreamer1.0-dev
sudo apt install python3.6 python3.6-dev
sudo apt install git autoconf automake libtool
sudo apt install python3-gi python-gst-1.0 
sudo apt install libgirepository1.0-dev
sudo apt install libcairo2-dev gir1.2-gstreamer-1.0

pip3 install --user pycairo PyGObject PyQt5
```

# EXECUTION

```bash
python3 gst-browser.py
```
